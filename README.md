# CoreDrive
Free NAS system in Django Framework.


# โปรดอ่านก่อนนำไปใช้งาน
ผลงานนี้มีความประสงค์เพื่อการศึกษาในห้องเรียน แต่ต้องการแบ่งปันสำหรับผู้คนที่ต้องการนำไปพัฒนาต่อ ซึ่งมีประโยชน์ในการใช้งานแบบ LocalHost กลุ่ม HomeServer เป็นหลัก ความสมบูรณ์ของระบบอยู่ที่ 90% สามารถนำไปใช้งานได้ทันที


# โปรแกรมที่ต้องติดตั้งก่อนใช้งาน
1. Python
2. Git


# ขั้นตอนการติดตั้ง
1. Git Pull ข้อมูลลงมาในคอมพิวเตอร์ทั้งหมดก่อน
2. เปิดใช้งาน env ใน Project
3. Run "python manage.py makemigration" เช็คความเปลี่ยนแปลงก่อน
4. Run "python manage.py migrate" เพื่ออัพเดทฐานข้อมูล
5. Run "python manage.py runserver"
